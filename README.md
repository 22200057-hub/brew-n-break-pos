# Brew N' Break — POS System

A web-based Point-of-Sale (POS) system built for **Brew N' Break**, a café and billiard shop. This system allows staff and administrators to manage café orders, billiard sessions, bookings, transactions, and generate reports — all from a single platform.

---

## Features

- **Dashboard** — Real-time overview of daily sales, revenue, billiard table status, and top-selling items
- **Café Management** — Browse menu, manage products, and track orders
- **Billiard Management** — Monitor table sessions, set reservations, and track session time
- **Transaction Management** — View and checkout café and billiard transactions
- **Bookings** — Manage Airbnb and room reservations
- **Reports** — Generate and print daily sales reports
- **Notifications** — Real-time alerts for expiring billiard sessions
- **Staff & Admin Roles** — Separate views and permissions for Admin and Staff users
